# Books & Authors SPA

A Vue 3 Single Page Application for managing books and authors with a public site and admin dashboard.

## Quick Start



# Install dependencies
npm install

# Terminal 1: Start API server
npm run server

# Terminal 2: Start Vue app
npm run dev
